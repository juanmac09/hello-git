commit -m Para hacer guardados hasta ese punto
reflog para ver todos los commit del repositorio
checkout para ir a uno de los puntos guardados
tag para marcar un punto importante de la aplicacion como una version
add para añadir un fichero al repositorio guardados
add . para añadir todos los archivos al repositorio guardado
Para movernos entre tag es git checkout tag/nombre del tag
git tag muestra los tag existentes
git checkout main volvemos al punto donde estamos
merge traer de cosas de una rama  a otra
stash nos sirve para guardar sin hacer commit 
git stash list para mostrar los stash 
git stash pop restaura el stash que hayamos hecho antes 
git stash drop borra los stash
git diff es para ver diferencias de ficheros o ramas 
brach muestra las ramas
brach nombre crea una rama nueva
branch -d nombre de rama para elminar ramas
